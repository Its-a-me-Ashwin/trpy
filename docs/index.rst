Introduction
------------
This library provides an easy-to-use Python SDK for controlling Trossen Robotics robot arms. 
It offers thread-safe function calls, ensuring seamless integration with AI agents while minimizing setup complexity. 
The SDK is designed to be efficient, reliable, and user-friendly, making it ideal for applications in automation, research, and robotics development.
